# ココナラサムネ Creator

## 使う場面

ユーザーが「ココナラのサムネ」「出品サービス用サムネ」を作りたい、または既存のココナラサムネを直したいと明示したときに使う。
汎用のサムネ制作は `skills/design/thumbnail-creator/SKILL.md` を使う。こちらはココナラ出品サムネに特化した反復修正ワークフロー。

## フォルダ規約

`projects/<サービス名>-thumbnail/` に以下を置く。

- `prompt.md` — 土台(v1)生成プロンプト
- `generate.mjs` — 土台生成スクリプト
- `references/` — 参考にする既存サムネ画像(構造・雰囲気抽出用。コピー/トレース禁止)
- `output/thumbnail.png` — 土台(v1)
- 修正のたびに追加:
  - `prompt-edit-<変更内容>.md`
  - `generate-edit-<変更内容>.mjs`
  - `output/thumbnail-v<N>-<変更内容>.png`

ファイル名の `<変更内容>` は "add-label" "remove-badge" のように何を変えたか一目でわかる短い英語スラッグにする(単なる連番だけにしない)。

## 手順

### 1. 土台(v1)を作る

1. 目的・訴求軸・ターゲット・トーン・必須コピー・参考画像の有無を確認する。
2. `skills/design/image-prompt-generator/SKILL.md` に従い `prompt.md` を作る。ココナラサムネは縦横比 **約6:5**(横1200×縦1000程度)が基本。
3. `generate.mjs` を書き、`server/codexImageClient.mjs` の `generateImageWithCodexAppServer` を使って生成する(`taskType: "showcase"`)。参考画像がある場合は `refImages` に渡す。
4. `output/thumbnail.png` に保存する。

### 2. 要素単位で反復修正する(このスキルの核心)

ユーザーから修正依頼が来るたびに、**画像全体を作り直さず、直前の画像を@img1として渡し、指定された要素だけを変える1回の編集**を行う。

1. 直前の最新出力(`output/thumbnail-v<N-1>-*.png` または `output/thumbnail.png`)を参照画像にする。
2. `prompt-edit-<変更内容>.md` を新規作成し、必ず以下の構成にする:
   - 【入力】: @img1 が今どんな状態か(現在確定している全要素)を簡潔に書く。
   - 【変更する要素(これだけ)】: 今回変えたい要素だけを具体的に書く。文言を変える場合は before→after を明記し、一行厳守・改行禁止などの制約も書く。
   - 【絶対に変更してはいけない要素】: 見出し・小見出し・バッジ・モデルの顔/ポーズ/髪型/服装/位置/大きさ・背景コラージュ・スマホモックアップ・全体構図・余白・縦横比など、変えてはいけない要素を1つずつ列挙する。
   - 【出力条件】: 「@img1と見比べて変わっているのは◯◯だけであること」「削除跡や不自然な余白・歪みは禁止」「高解像度でそのまま使える仕上がり」を必ず入れる。
3. `generate-edit-<変更内容>.mjs` を作る(既存の `generate-edit-*.mjs` をコピーして `refImages` の参照先パスと `imageName`/`sectionId`/出力ファイル名だけ差し替えれば良い)。
4. スクリプトを実行し、`output/thumbnail-v<N>-<変更内容>.png` に保存する。
5. 生成結果を確認し(Read tool で画像表示、必要なら `open <path>` でプレビューアプリでも開く)、変更点を一言で報告して次の指示を待つ。

### 3. 確定

ユーザーが確定と言ったら、最終ファイル名を伝える。必要なら案件フォルダ、`portfolio/`、`exports/` へコピーする。

## 注意

- 1回の編集プロンプトで複数要素を同時に変えない。要素ごとに分けて1つずつ確実に反映する。
- 参考画像・既存サムネの構造は流用してよいが、コピー文言・実在人物・ロゴ・既存デザインのトレースは禁止。新しい文言に書き換える。
- 文字切れ、可読性、小さい表示サイズでの視認性、四辺の安全マージンを毎回確認する。
- 削除系の編集(バッジ・ラベルなど)は、削除跡の不自然な余白や輪郭が残らないよう、背景を自然に埋め直す指示を必ず入れる。
