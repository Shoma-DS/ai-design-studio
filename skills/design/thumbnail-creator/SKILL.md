# サムネ Creator

## 使う場面

ユーザーが「サムネ」を作りたいと明示したときに使う。

## 手順

1. 目的、媒体、サイズ、ターゲット、トーン、必須コピーを確認する。
2. 明示されていない重要条件だけ質問する。
3. コピーと構成を作る。
4. `skills/design/image-prompt-generator/SKILL.md` に従って画像プロンプトを作る。
5. Codex app-server / gpt-image-2 でラスター画像を生成する。
6. 成果物を案件フォルダ、`portfolio/`、または `exports/` に保存する。

## 注意

- 参考画像は構造・雰囲気・視線誘導のみ使う。
- 商標、実在人物、元コピーの再現は禁止。
- 文字切れ、可読性、余白、CTAの視認性を確認する。
