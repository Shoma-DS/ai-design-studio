# LP バージョン作成スキル

## 概要

LP原稿を新しいバージョンフォルダにコピーし、
指定されたビジュアルスタイル（文字で渡された仕様 or 参考画像）に合わせた
セクション別画像プロンプト（nanobanana pro形式）を生成・保存する。

---

## 入力として受け取るもの

| 入力 | 必須 | 説明 |
|------|------|------|
| LP原稿パス | 必須 | コピー元の `LP原稿.md` のrepo内パス |
| ビジュアルスタイル文字列 | 必須 | デザインテーマ名・配色・雰囲気・書体の説明テキスト |
| 参考画像 | 任意 | 雰囲気を参照する画像（添付または `@img` パス） |
| バージョン名 | 必須 | フォルダ名に使う識別子（例: `custom-style-draft`） |
| 出力先ベースパス | 必須 | 例: `projects/ai-income-course/` |

---

## 実行ステップ

### Step 1 — バージョンフォルダ作成・LP原稿コピー

```
1. {出力先ベースパス}/{バージョン名}/ フォルダを作成
2. LP原稿.md を新フォルダにコピー
```

### Step 2 — ビジュアルスタイルの解析

ユーザーが渡したビジュアルスタイル文字列・参考画像から以下を抽出する：

- **背景テクスチャ・ベースカラー**（例: 和紙・大理石・グリーン）
- **アクセントカラー**（HEXコード付きで）
- **見出し書体の雰囲気**（筆文字・セリフ・サンセリフ 等）
- **デコレーション要素**（金継ぎライン・墨のかすれ・大理石のベイン 等）
- **全体トーン**（エレガント・禅・清潔・高級 等）
- **禁止事項**（使ってはいけない色・表現 等）

### Step 3 — セクション別プロンプト生成

LP原稿の各セクションに対して、nanobanana pro形式のプロンプトを生成する。

**対応セクション（LP原稿の構成に合わせて増減）：**

| ID | セクション名 |
|----|------------|
| SEC-01 | HERO（ファーストビュー） |
| SEC-02 | PAIN（こんなことを感じていませんか？） |
| SEC-03 | WHY NOW（今、動くべき理由） |
| SEC-04 | WHAT IS AGENT（違いの説明） |
| SEC-05 | CASES（事例・成果） |
| SEC-06 | TESTIMONIALS（受講者の声） |
| SEC-07 | CURRICULUM（講座内容） |
| SEC-08 | BENEFITS（特典） |
| SEC-09 | CAMPAIGN（限定オファー） |
| SEC-10 | DIFFERENCE（他と違う点） |
| SEC-11 | COMPARISON（比較表） |
| SEC-12 | FAQ（よくある質問） |
| SEC-13 | FINAL CTA（最終参加案内） |

### Step 4 — プロンプトファイル保存

生成したプロンプトを `{バージョンフォルダ}/image-prompts.md` に保存。

---

## nanobanana pro プロンプトの必須構造

各セクションのプロンプトは必ずこの構造で作る：

```
あなたは【トップクラスのインフォグラフィックデザイナー兼グラフィックレコーダー】です。

目的：
{{セクションの役割と視覚化の目的}}

────────────────
【入力】
@img（なし：ゼロベース新規生成）
　または
@img（参考画像）

────────────────
【このプロンプト単体で必ず守る共通デザイン方針】

・アスペクト比は9:16、縦型、1080×1920px想定
・{{ビジュアルスタイルから抽出した背景・カラー・書体・装飾のルール}}
・全体の雰囲気は {{トーン}}
・禁止：{{禁止スタイル}}
・スマホで見た瞬間に情報が入る視認性を最優先

────────────────
【最重要ルール（絶対に守ること）】

{{セクション固有の制約 3〜5項目}}

────────────────
【構図】

{{レイアウト説明}}

視線誘導：
① → ② → ③

────────────────
【タイトル・コピー】

{{LP原稿から該当セクションのコピーをそのまま記載}}

────────────────
【色・雰囲気ルール】

{{配色の具体的な使い分け（HEXコード付き）}}

────────────────
【デザイン】

{{スタイルのコンセプト一言・デザイン哲学}}

────────────────
【出力条件】

・9:16縦型
・{{セクション固有の出力条件}}

────────────────
【最終目的】

見た瞬間にこう思わせる：
「{{ユーザーに与えたい感情・アクション}}」
```

## 共通デザイン方針の書き方（ビジュアルスタイル別テンプレ）

### Marble / White / Gold テンプレ
```
・背景は白大理石（Ghost White #F8F8FF）に金の亀裂ライン（Goldenrod #DAA520）が走るテクスチャ
・見出し書体は細いセリフ体（Didot・Bodoni 風）
・テキスト色は黒 #000000 / ゴールド #DAA520
・全体の雰囲気は高級・洗練・冷涼・エレガント
・フラットイラスト風・AI生成イラスト風は禁止
```

### Calligraphy / Ink Wash / Gold Leaf テンプレ
```
・背景は和紙テクスチャ（繊維質、微妙な凹凸感）
・墨絵の筆致（かすれ・濃淡）と金箔フレークをアクセントに使う
・見出し書体は筆文字（ブラシカリグラフィー）
・主文字色：墨黒 #000000 / 金箔 #D4AF37 / 朱 #CC3333（印鑑的用法のみ）
・円相（Enso）・筆のかすれ・金箔フレークで装飾
・全体の雰囲気は禅・伝統・力強い・洗練
```

### Fresh Green / White テンプレ
```
・背景は白（#FFFFFF）またはライトグリーン（#E8F8EE）
・アクセントはフレッシュグリーン（#2ECC71）とミントグリーン（#A8E6CF）
・見出し書体はモダンサンセリフ
・全体の雰囲気は爽やか・清潔・モダン・親しみやすい
・リクルート誌スタイルのリアル写真ベース
```

## ファイル出力構成

```
{出力先ベースパス}/
└── {バージョン名}/
    ├── LP原稿.md         ← コピーしたもの（内容そのまま）
    └── image-prompts.md  ← 全セクションのnanobanana proプロンプト
```

## 実行例

```
入力：
- LP原稿: projects/ai-income-course/LP原稿.md
- スタイル: "Calligraphy / Ink Wash / Gold Leaf"（テキスト仕様 + 参考画像）
- バージョン名: custom-style-draft
- 出力先: projects/ai-income-course/

実行：
1. custom-style-draft/ フォルダ作成
2. LP原稿.md コピー
3. Calligraphy スタイルで全13セクションのプロンプト生成
4. image-prompts.md に保存
```
