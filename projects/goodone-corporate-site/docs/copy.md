# 原稿

## トップページ

メインコピー: 業務車両の稼働を止めない、次世代エンジンクリーニング。

リード: DPFアッシュクリーニングと水素カーボンクリーニングを軸に、運送・建設・整備の現場が抱えるコスト、時間、黒煙対策を一体で支援します。

課題: DPFやインジェクター交換の高額化、修理による長期停止、黒煙・白煙、代理店商材の不足。

## 会社概要

株式会社グッドワンは、ERC製カーボンクリーニングシステムの日本輸入総代理店として、業務車両向けメンテナンス、DPFアッシュクリーニング、水素カーボンクリーニングを展開しています。

## サービス紹介

DPFアッシュクリーニング、水素カーボンクリーニング、ERCクリーナー供給、代理店・施工店サポートを、法人が比較検討しやすいサービス体系として整理します。

## 実績紹介

運送会社、建設会社、整備工場などの現場課題ごとに、導入目的と期待結果を分けて掲載します。

施工相談、資料請求、代理店導入のご相談を受け付けます。車両の症状、台数、導入検討内容をお知らせください。
