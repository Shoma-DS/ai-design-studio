# セクション別画像プロンプト

画像生成はPC版とSP版の構図を分ける。参考サイトや実在ロゴ・写真のコピーは禁止し、構造・雰囲気・視線誘導のみ参考にする。

## hero-pc / hero-sp

あなたはトップクラスのBtoBテクニカルビジュアルデザイナーです。
目的: ERC日本輸入総代理店の信頼感と業務車両メンテナンスの専門性を伝えるコーポレートHPのファーストビュー背景。
入力: なし。ゼロベース新規生成。
最重要ルール: 実在企業ロゴや車両メーカーの商標を入れない。文字は入れない。
構図: PCは横長で、左側にコピーが載る余白を確保し、右側に整備ピット、業務用トラック、診断機器、清潔な工具を配置。SPは縦長で中央上部に車両と整備士、下部に余白。
色: 濃紺、白、赤アクセント。高品質で清潔。

## diagnostic-pc

目的: DPF差圧診断と短時間施工を示す説明セクション用画像。
構図: 診断メーター、ホース、整備士の手元、清潔なピット。数値やロゴ文字は入れない。

## service-pc

目的: DPF洗浄、水素カーボンクリーニング、製品供給の3要素を抽象的に表現するサービスページ画像。
構図: エンジン断面風の抽象パーツ、流体ライン、整備設備。写真調とインフォグラフィック調の中間。

## case-1 / case-2 / case-3

目的: 導入事例カード用画像。
構図: 運送会社の車両ヤード、建設現場の重機、整備工場の診断作業をそれぞれ写実的に表現。実在ロゴ、ナンバー、人物の個人特定要素は入れない。
