# ContactSection

**役割**：問い合わせ導線。メール／電話の2ボタンと、オフィス写真を組み合わせる。シンカのCONTACTセクション（ボタン2種＋動画サムネ）を踏襲しつつ、動画は使わず静止画に簡略化。

**レイアウト**
- モバイル：縦積み（コピー→画像、ボタンも縦積み）。
- 1024px以上：横並び（コピー側`flex:1`／画像`flex:0 0 40%`、画像比率`1/1`）、ボタンも横並び。

**配色・タイポ**：見出し`clamp(28px,7vw,40px)`900。プライマリボタンはamber-600背景・白文字、セカンダリはネイビー枠のアウトライン。両方hover時`translateY(-2px)`。

**リンク実装**：メールは`mailto:info@tsunagu-logistics.example.co.jp`（ダミー）、電話は`tel:0924412200`（`companyInfo.tel`から動的生成、ハイフン除去）。

**アニメーション**：コピーは`whileInView`で`y`フェードイン、画像は`scale:0.96→1`＋`opacity`フェードイン（delay 0.15s）。

**表示コンテンツ**：見出し「お問い合わせ」、リード文、ボタン2種（`docs/copy.md`参照）。
