# NumbersSection（TSUNAGU'S NUMBERS 数字で見るツナグ物流）

**役割**：統計データをカード形式で見せる。シンカの「数字でわかるシンカ」セクションに相当。イラストはAI画像生成を使わず、軽量な自作SVG（棒グラフ＋折れ線）で代替し、スコープと読み込みコストを抑えている。

**レイアウト**
- 背景`--color-offwhite`。モバイルはカード1カラム、561px以上2カラム、1024px以上4カラム。
- ヘッダー部（アイキャッチ＋見出し＋SVGチャート）は1024px以上で左右分割（`justify-content:space-between`）。

**アニメーション**
- 見出し群：`whileInView`フェードイン（duration 0.6〜0.7s）。
- 統計カード：`staggerChildren:0.12`で1枚ずつ`opacity:0,y:24→opacity:1,y:0`。
- 数値：`Counter`コンポーネントによるカウントアップアニメーション（ビューポート進入をトリガーに0→目標値へ増加）。

**表示コンテンツ**：`stats`配列4件（拠点数48／配送件数32,000件/日／ドライバー数1,200名／CO2削減率18%、`docs/copy.md`参照）。
