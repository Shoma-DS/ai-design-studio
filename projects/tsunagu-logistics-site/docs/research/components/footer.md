# Footer

**役割**：会社情報・ナビ再掲・コピーライト。シンカのフッター構成を簡略化して踏襲（シンカは背景グリーンのCTA帯＋白背景フッターの2段構成だが、本サイトはCtaBandと分離しフッター自体はシンプルな白背景に統一）。

**レイアウト**：モバイルは縦積み。1024px以上は`.top`（ロゴ＋会社名／住所)が横並び・両端揃えになる。ナビはflex-wrapで折り返し。

**配色・タイポ**：白背景、`--color-border`の上罫線区切り。文字は11〜12px中心の控えめなサイズ。

**表示コンテンツ**：ロゴ＋会社名、住所（〒812-0011 福岡県福岡市博多区博多駅前3丁目2-1）、電話・FAX、`navLinks`再掲、コピーライト表記（`docs/copy.md`参照）。
