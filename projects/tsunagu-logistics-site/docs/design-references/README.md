# design-references について

このフォルダは、参照サイト（シンカ株式会社）と自サイト（ツナグ物流）のスクリーンショットを保存する想定でしたが、
現在の自動化ツール環境では `save_to_disk` で撮影したスクリーンショットの実ファイルパスをこのマシン上から特定できず、
画像ファイルの保存を完了できていません（2回の検証セッションで再現、環境側の制約と判断）。

代替として、参照サイトの構成・配色・アニメーション等の分析結果は `../research/analysis.md` にテキストで詳細にまとめてあります。
スクリーンショットが必要な場合は、`https://shinka-inc.com/` と `http://localhost:3000/`（本サイトのローカル起動時）を
手動でブラウザで開いて直接比較することを推奨します。
