# 最短でAI副業を成功させる方法！AI副業の始め方講座 LP原稿 v3

## 制作方針（FVコピー連動版）

- FVの主張を「会社の昇給だけを待つのはやめ、AIを使った新しい働き方の選択肢を持つ」に統一。
- LP全体の流れを、昇給待ちの不安 → 国データで見る家計環境 → 副業・AI案件の増加 → 案件化の順番 → 60分講座で最初のルートを決める、に再構成。
- 「AIを使える」ではなく「AIを使って納品し、収入に変える」を軸にする。
- CTAは「60分の無料講座に申し込む →」へ統一。
- 特典はFV下部オファーと同じく「AI副業ロードマップ」を中心に据える。
- 不安訴求は煽りすぎず、時間がない会社員・子育て中・AI使用中だが未収益の人が自分ごと化できる表現にする。
- 各セクションの締めは「まず自分に合う最初のルートを確認する」へ戻す。
- データ訴求は、厚生労働省・総務省などの公的データと、クラウドワークス・ランサーズ等の公開情報を分けて扱う。

「何をAIに任せるか」「何を納品物にするか」「誰にいくらで提案するか」まで決まって、初めて仕事になります。

出典: NTTドコモ モバイル社会研究所「生成AI利用率、過半数に。1年で急増」

**締め**
足りないのは、才能でも最新ツールでもありません。
AIを収入に変える順番です。
