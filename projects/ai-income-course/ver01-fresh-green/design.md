# 最新AI活用 完全マスター講座 LP Design Guide

## 基準

- 基準画像は `lp/images/IMG_hero_bg.png`。
- FVと同じ「明るい実写背景 + 太いコピー + 手書き文字 + フレッシュグリーンCTA + 金の筆ライン」の品質感に揃える。
- 画像は「素材を貼っただけ」にしない。FVのように、見出し、人物/手元/背景、カード、CTA、筆ライン、余白が1枚の広告として設計されている状態を合格基準にする。
- `web-sozai.com` の素材は主役ではなく、背景の上に置く透明PNGステッカーとして扱う。
- ロボット、AI顔、回路脳、青い発光UI、サイバー感の強い素材は使わない。
- 減らす対象はリアル人物写真やリアルな広告モデル風人物の「多さ」。自然な配置であれば、広告モデル風の人物やリアル寄り人物を使ってよい。
- web-sozai由来の人物イラストは使ってよい。説明用・装飾用なら複数でも可。ただし同じ素材の反復や、人物イラストを画面の主役にしすぎることは避ける。
- リアルな広告モデル人物を使う場合は1セクション1人までを基本にし、主役コピーや図解を邪魔しない位置に置く。手元、横顔、後ろ姿、小さめの講師/受講者表現が合う。

## 色の禁止

- `#006B3A` より暗い緑を大面積に使わない。
- 青系AIカラー、web-sozai由来の青い線、ネオン、紫グラデーション、黒背景、サイバー背景は禁止。
- 上部の大きな黒枠、黒い帯、黒い矩形見出し、黒背景のフレームは禁止。
- 緑一色に寄せず、白・金・黒・淡いミントで抜けを作る。

## Typography

- 大見出しはFVの「昇給待ち、やめました！」級の太さと迫力にする。
- 重要語は黒文字 + 緑文字の混植で、FVのように動きを出す。
- 見出しは黒い帯に乗せず、明るい背景の余白に大きな黒文字として置く。必要なら白い半透明ラベル、金の筆ライン、緑の下線で締める。
- 手書き文字は各セクションに1箇所だけ入れる。例: 「まずは順番だけでOK」「ここが分かれ道」「ムリなく整理」。
- 小さい説明文を詰め込まない。スマホで読める行数を優先する。

## Layout System

- 1080 x 1919px、スマホ縦長。
- 上部: 手書きコピーまたは小バッジ。
- 中上部: 太いメイン見出し。黒枠ではなく、白余白、白い半透明ラベル、金ライン、緑強調でFV級の視認性を出す。
- 中央: 1つの主役図解。カードは最大3枚まで。
- 下部: グリーンCTA帯または次セクションへつながる締めカード。
- 余白を先に確保し、その余白にweb-sozaiステッカーを配置する。
- 情報量が多いデータは1枚に詰めない。`SEC-03`, `SEC-04`, `SEC-05` に分割し、各画像の主役メッセージは1つだけにする。

## web-sozai Overlay Rules

- 背景を先に作る。その上に素材PNGを置く前提で、空白・カード端・吹き出し横に配置する。
- 素材は10〜28度だけ回転、少し影を付ける、または白縁を付けてステッカー感を出す。
- 素材サイズは画面幅の12〜28%。主役コピーより大きくしない。
- 素材を画面中央に大きく置かない。端、カード角、見出し横、CTA周辺の装飾として使う。
- 同じセクション内で同じweb-sozai素材を繰り返し使わない。1素材1回まで。
- 素材の元色が青い場合、LP上には青を残さず必ずブランドグリーンまたは金へ置き換える。置き換えられない場合は、その素材を使わない。
- `websozai-documents`, `websozai-graph-up`, `websozai-pc-woman`, `websozai-meeting-explain`, `websozai-meeting-two`, `websozai-discussion` を使う。
- `websozai-online-meeting` の4人グリッドはセミナー感が強いため、個別相談・ヒアリング・無料講座の導入表現では使わない。
- 無料講座の「受講する」表現では、PC外の人物は受講者1人だけにし、相手はPC画面内にだけ置く。対面で同じ机に座る2人、横並びで同じPCを見る2人、PC外の講師は入れない。
- `websozai-ai-*` は使わない。

## Person Rules

- web-sozai由来の人物イラストは使ってよい。自然な装飾や説明なら複数入れてもよいが、同一素材を繰り返さない。
- 広告モデル風のリアル人物やリアル寄り人物は使ってよい。ただし自然な仕事・学習・相談シーンとして扱う。
- リアル人物は1セクション1人までを基本にする。会議や相談表現でも小さく2人まで。
- リアル人物のカメラ目線の決めポーズ、大げさな笑顔、人物集合、肌が整いすぎたAI写真感は禁止。
- データ訴求のセクションでは、人物よりも数字、資料、手元、生活背景を優先する。

## Data Story Split

- `SEC-03 PAIN`: 生活不安と副業の現実。JILPT「副業の月収は5万円未満が4割強」を1カードで使う。
- `SEC-04 WHAT IS AI`: ChatGPTを触ることと、仕事に変えることの違い。NTTドコモ モバイル社会研究所「仕事・学業で生成AI利用20%」を小カードで使う。
- `SEC-05 WHY NOW`: 需要は高いが、使える人が足りない。経産省系推計「2030年AI人材不足 最大12.4万人」、NRI「生成AI活用課題 リテラシー/スキル不足70.3%」を2カードに分けて使う。
