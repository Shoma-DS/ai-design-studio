# 09-access（ACCESSカード：アクセス情報の背景）

共通ルール（全セクション共通）:
- 縦型9:16、1080×1920px相当のスマホ縦画面いっぱいに写る構図
- 写真（実写風）のみで、文字・ロゴ・グラフィック装飾・キャプションは一切焼き込まない（テキストは後からHTMLで重ねる）
- 商標・実在人物・実在ブランド・実在サロンの模写は禁止、完全にオリジナルの人物・空間・デザインとして生成する
- 広告然としすぎない、上品で洗練された自然光の写真トーン

## プロンプト

【重要・アスペクト比】縦長ポートレート写真。幅1080px×高さ1920px（width < height、9:16の縦長）で出力する。横長（landscape）やほぼ正方形の構図は禁止。カメラを縦に構えた、上下方向に長い構図にする。

上品なプライベートネイルサロンの入口・外観。落ち着いたベージュ〜ブラウンの外壁、控えめな木製ドア、小さな観葉植物やシンプルなサインプレート（文字は入れない）、夕方の柔らかい自然光。ドアを画面中央〜上部に、下部に石畳やアプローチが縦方向に伸びる構図にし、横に広がるパノラマ的な構図にしない。画面下部に住所・営業時間テキストを重ねられる余白を作る。
