# 制作タイプ確認ワークフロー

## 目的

ユーザーが作りたいものを明示していない場合、制作タイプを確認してから作業を始める。

## 質問

```text
今回作るものは、バナー、サムネ、SNS投稿、スライド、LPのどれですか？
```

## ルーティング

- バナー: `workflows/banner.md`, `skills/design/banner-creator/SKILL.md`
- サムネ: `workflows/thumbnail.md`, `skills/design/thumbnail-creator/SKILL.md`
- SNS投稿: `workflows/sns-post.md`, `skills/design/sns-post-creator/SKILL.md`
- スライド: `workflows/slide.md`, `skills/design/slide-creator/SKILL.md`
- LP: `workflows/lp.md`, `skills/design/lp-creator/SKILL.md`
