# Site Animations（実サイト分析パターン集）

実在サイトのアニメーション実装を解析し、技法（構造・タイミング・イージング）だけを抽出して汎用CSS/JSに書き直したパターン集。`animate-css`・`hover-css`（第三者ライブラリのクローン）とは別に、**ライブラリのプリセットでは足りない挙動**（スクロール連動の帯ワイプ演出、marquee風の無限ループ、ヘッダー縮小など）を持ち込むために使う。

## 使う場面

`skills/design/animate-css/SKILL.md` から参照される。実装したい演出がAnimate.cssのキーフレーム一覧に無い場合、ここのカタログを先に確認する。

## 収録方針

- 元サイトのコードそのものではなく、**挙動の技法**（distance/duration/easing/delayの数値感、DOM構造、キーフレームの組み方）を汎用化して記録する。
- 元サイトのコピー・ロゴ・配色・商標は再現しない（`references/README.md`の方針と同じ）。
- ファイル名は解析元ドメインを付ける（例: `enatural-co-jp.md`）。どのサイトから抽出したかを常に明記する。
