# 見積書テンプレート（LP / HP）

クライアント向けご提案書・お見積り書のHTMLテンプレート。`黄色 シンプル Webサイト提案書` デザイン（他社サンプル資料）の構成・レイアウトを参考に、自社用に再構築したもの。文言・ロゴ・配色はそのまま流用せず、目的（構成の参考）のみ利用している。

## ファイル

- `lp-quote-template.html` — LP（1枚絵ラスターLP／スワイプ型LP含む）向け
- `hp-quote-template.html` — HP（Next.js実装のコーポレートサイト）向け

どちらもCSS・JSを内蔵した単体HTMLで、外部ファイルへの依存はない（コピーして1ファイルだけ渡せる）。

## 自動生成

「見積書を作成して」「◯◯様の見積書作って」と頼めば `skills/common/quote-creator/SKILL.md` が動き、該当テンプレートを `exports/quotes/` へコピーして分かっている情報を埋めた状態で渡してくれる。このフォルダのテンプレート自体は直接編集しない。

## 使い方（手動で使う場合）

1. 案件ごとにテンプレートをコピーして使う（このファイル自体は直接編集しない）
2. ブラウザで開くと、本文の文字は**クリックしてそのまま書き換えられる**（`【クライアント名】` などのプレースホルダーも含む）。マウスを乗せると薄い黄色でハイライトされる箇所が編集可能な範囲。
3. 右上のツールバーから:
   - 💾 編集内容を保存 — 今の編集状態をHTMLファイルとしてダウンロードする（クライアント名+日付のファイル名になる）
   - 🖨 印刷 / PDF化 — その場でPDF化する。設定は以下推奨:
     - 用紙の向き: 横（landscape）
     - 余白: なし
     - 背景のグラフィックス: ON（黄色ヘッダー帯などを再現するため）
4. 課題・目的・構成案・スケジュール・見積り項目の文言も同様にクリックして案件に合わせて書き換える

## LP用とHP用で分けている理由

- LP：1枚絵の縦スクロールLP。見積り項目は「構成設計・原稿・デザイン（画像生成）・レスポンシブコーディング・公開」中心で、CMS構築は含まない。金額感は数万〜数十万円規模。
- HP：Next.js実装の実コードサイト。見積り項目は「企画設計・デザイン・実装・CMS構築（任意）・公開」で、複数ページ・サイトマップが前提。金額感はLPより大きく、100万円前後まで想定。

金額は目安。実際の提示額はページ数・機能要件・案件ごとの相場感に応じて調整する。

## 現状分析スライドについて

既存サイト／既存LPのリニューアル案件でのみ使用する。新規制作の場合はスライドごと削除してよい。
